# 🎆 ASCII 花火ジェネレーター

ターミナルで花火がリアルタイムにアニメーションする Python スクリプトです。

## 必要なもの

- Python 3.9 以上
- Windows 10 / 11（ANSI カラー対応）
- ターミナルを全画面にするとより綺麗です！

## 使い方

```bash
python hanabi.py
```

## 操作方法

| キー | 動作 |
|------|------|
| `Enter` / `Space` | 新しい花火を打ち上げる |
| `Q` / `Esc` | 終了 |

## 特徴

- 🌟 星空の背景
- 🎨 12 色のカラーパレット
- 💥 多色爆発（ランダムで 40% の確率）
- ✨ ロケットの尾・爆発の残光・流れ星エフェクト
- 🚀 複数の花火を同時に打ち上げ可能（最大 12 発）
- ⚡ 自動打ち上げ機能（花火がないときに自動で上がる）
